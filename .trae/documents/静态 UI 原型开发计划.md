# 静态 UI 原型开发计划

收到，接下来的所有文档、待办事项和沟通将全部使用中文。
为了让您直观地体验 UI 交互，我将开发一个基于 Vue 3 + Element Plus 的高保真静态原型。

## 1. 项目初始化
*   **创建项目**: 在 `health-manage-frontend` 目录下初始化 Vue 3 + Vite 项目。
*   **安装依赖**:
    *   `element-plus`: 用于构建美观、现代的 UI 组件。
    *   `echarts`: 用于绘制健康趋势图表。
    *   `vue-router`: 管理页面路由跳转。
    *   `sass`: 编写 CSS 样式。
    *   `@element-plus/icons-vue`: 界面图标库。

## 2. 页面开发详细规划

### (1) 全局布局 (Layout)
*   **侧边栏**: 包含“仪表盘”、“健康档案”、“打卡计划”、“个人中心”导航。
*   **顶部栏**: 显示用户信息、消息通知和退出按钮。

### (2) 登录/注册页 (Login)
*   **设计**: 时尚的左右分栏布局。左侧为健康生活主题插画，右侧为极简交互表单。
*   **交互**: 输入框聚焦动画、按钮波纹效果。

### (3) 仪表盘 (Dashboard)
*   **核心指标**: 展示心率、血压、体重、步数的实时卡片。
*   **图表可视化**: 集成 ECharts，展示“体重变化趋势”和“血压监测”曲线。
*   **快捷操作**: “记一笔”和“今日打卡”快速入口。

### (4) 健康数据管理 (Health Data)
*   **功能**: 历史数据列表展示。
*   **交互**:
    *   **新增**: 点击“+”弹出毛玻璃效果模态框录入数据。
    *   **删除**: 列表项右侧提供删除按钮，带确认提示。

### (5) 打卡管理 (Check-in)
*   **设计**: 卡片式网格布局展示打卡习惯（如：晨跑、吃水果）。
*   **交互**: 点击卡片即完成打卡，伴随颜色变化和微动画。
