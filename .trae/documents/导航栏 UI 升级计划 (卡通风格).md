# 导航栏 UI 升级计划 (卡通风格)

收到您的反馈，现有的深色侧边栏确实比较商务传统。为了营造“好看、时尚、偏卡通”的风格，我将对 `Layout.vue` 进行大改版。

## 1. 设计思路
*   **配色**: 摒弃深色背景，改用 **清新的浅色系** 或 **渐变色**。例如：白色圆角卡片悬浮在浅灰背景上，选中项使用高饱和度的糖果色（如薄荷绿、蜜桃粉）。
*   **形状**: 大量使用 **圆角 (Border Radius)**。侧边栏不再是直角的，而是悬浮的圆角矩形。
*   **图标**: 增大图标尺寸，选中时增加 **Q弹动画**（如缩放或跳动）。
*   **布局**: 尝试 **“悬浮侧边栏”** 风格，让侧边栏与主内容区分隔开，更像一个独立的控制面板。

## 2. 具体修改方案
1.  **背景**: 整个页面背景设为柔和的浅灰色 (`#f0f2f5`)。
2.  **侧边栏 (Sidebar)**:
    *   变为白色背景，带有圆角和轻微阴影。
    *   与屏幕左侧保留一定间距（悬浮感）。
    *   Logo 区域设计得更活泼。
3.  **菜单项 (Menu Item)**:
    *   选中状态：背景变为渐变色（如浅绿渐变），文字变白，图标变大。
    *   未选中状态：灰色图标，鼠标悬停时有轻微背景色。
    *   增加过渡动画 `transition: all 0.3s cubic-bezier(...)`。

*   调整 `src/styles/main.scss` 中的全局背景色。
