# 静态 UI 原型开发计划

为了让您直观地体验 UI 交互，我将优先开发前端静态页面原型。这将是一个基于 Vue 3 + Element Plus 的可交互演示版本，后续可以直接接入后端接口。

## 1. 项目初始化 (Project Setup)
*   **创建项目**: 在 `health-manage-frontend` 目录下初始化 Vue 3 + Vite 项目。
*   **安装依赖**:
    *   `element-plus`: 用于构建美观的 UI 组件。
    *   `echarts`: 用于绘制健康数据图表。
    *   `vue-router`: 用于页面跳转。
    *   `sass`: 用于编写 CSS 样式。
    *   `@element-plus/icons-vue`: 图标库。

## 2. 页面开发 (Page Implementation)

### (1) 全局布局 (Layout)
*   实现包含 **侧边导航栏 (Sidebar)** 和 **顶部通栏 (Header)** 的主框架。
*   侧边栏菜单包括：首页、健康档案、打卡计划、个人中心。

### (2) 登录/注册页 (Login/Register)
*   **路径**: `/login`
*   **内容**: 实现左右分栏设计的静态页面。左侧放置时尚插画，右侧放置登录/注册表单（带浮动标签交互效果）。

### (3) 首页仪表盘 (Dashboard)
*   **路径**: `/dashboard`
*   **内容**:
    *   **顶部卡片**: 展示心率、血压、体重、步数等模拟数据。
    *   **图表区**: 集成 ECharts，展示“近7天体重变化”和“血压趋势”的静态图表。
    *   **快捷入口**: 放置“快速记一笔”和“今日打卡”按钮。

### (4) 健康数据管理页 (Health Data)
*   **路径**: `/health-data`
*   **内容**:
    *   **数据表格**: 使用 Element Plus 表格展示模拟的历史健康记录。
    *   **交互**: 点击“新增”按钮弹出毛玻璃效果的模态框；点击“删除”图标弹出确认气泡。

### (5) 打卡管理页 (Check-in)
*   **路径**: `/check-in`
*   **内容**:
    *   使用卡片网格布局展示打卡任务（如：晨跑、吃水果）。
    *   **交互**: 点击卡片模拟“打卡成功”的颜色变化和动画效果。
